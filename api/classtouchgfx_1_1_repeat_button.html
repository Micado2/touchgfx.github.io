Detailed Description

A RepeatButton is similar to a regular Button, but it will 'repeat' if pressed for a long period of time.

The RepeatButton differs from a regular Button with regards to activation. A Button is activated when the button is released, whereas a RepeatButton is activated immediately when pressed and then at regular intervals. A RepeatButton does not activate when released.

As for other well-known repeat buttons, the interval from the first activation until the second activation as well as the subsequent interval between activations can be set for the RepeatButton.

The default values for initial delay is 10 ticks, and the default value for the following delays between button activations is 5 ticks.
