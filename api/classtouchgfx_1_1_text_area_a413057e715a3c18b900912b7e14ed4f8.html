void setWideTextAction ( WideTextAction  action)
inline

Sets wide text action. Defines what to do if a line of text is wider than the text area. Default action is WIDE_TEXT_NONE which means that text lines are only broken if there is a newline in the text.

If wrapping is enabled and the text would occupy more lines than the size of the TextArea, the last line will get an ellipsis to signal that some text is missing. The character used for ellipsis is taken from the text spreadsheet.

Parameters
actionThe action to perform for wide lines of text.
