Detailed Description

Using Line from CanvasWidgetRenderer, progress will be rendered as a line.

This means that the user must create a painter for painting the circle. The line does not need to horizontal or vertical, but can start at any coordinate and finish at any coordinate.

Note
As LineProgress uses CanvasWidgetRenderer, it is important that a buffer is set up by calling CanvasWidgetRendere::setBuffer().
