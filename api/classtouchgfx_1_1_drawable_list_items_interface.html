Detailed Description

A drawable list items interface.

Used to pass the allocated array of drawable elements to ScrollList::setDrawables(), ScrollWheel::setDrawables() or ScrollWheelWithSelectionStyle::setDrawables(). Provides easy access to each element in the array as well as the size of the array.

See also
ScrollList::setDrawables, ScrollWheel::setDrawables, ScrollWheelWithSelectionStyle::setDrawables
