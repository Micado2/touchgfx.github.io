Detailed Description

This class is an abstract interface for a class allocating partial framebuffer blocks.

The interface must be implemented by a subclass.

See also
SingleBlockAllocator, ManyBlockAllocator

)
pure virtual

Allocates a framebuffer block.

The block will have at least the width requested. The height of the allocated block can be lower than requested if not enough memory is available.

Parameters
xThe absolute x coordinate of the block on the screen.
yThe absolute y coordinate of the block on the screen.
widthThe width of the block.
heightThe height of the block.
[in,out]blockPointer to pointer to return the block address in.
Returns
The height of the allocated block.

Implemented in ManyBlockAllocator< block_size, blocks, bytes_pr_pixel >, and SingleBlockAllocator< block_size, bytes_pr_pixel >.
