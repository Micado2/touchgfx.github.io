Attaches a buffer. Can be used if the buffer is not ready when the Rendering buffer is created initially.

Parameters
[in]buf_Pointer to the frame buffer where the image is rendered.
xAdjust_Horizontal adjustment of the x coordinate, used when bits per pixel is less than eight which implies that a uint8_t pointer cannot precisely address the start of the frame buffer.
width_The width of the frame buffer to write.
height_The height of the frame buffer to write.
stride_How much to add the a pointer inside the frame buffer to advance to the next line in the frame buffer.
