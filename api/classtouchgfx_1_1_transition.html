Detailed Description

The Transition class is the base class for Transitions.

Implementations of Transition defines what happens when transitioning between Screens, which typically involves visual effects. An example of a transition implementation can be seen in example custom_transition_example. The most basic transition is the NoTransition class that does a transition without any visual effects.

See also
NoTransition, SlideTransition
