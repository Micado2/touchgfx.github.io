Detailed Description

template<class T>
class touchgfx::Snapper< T >

A mix-in that will make class T draggable and able to snap to a position when a drag operation has ended.

The mix-in is able to perform callbacks when the snapper gets dragged and when the Snapper snaps to its snap position.

◆ handleClickEvent()

virtual void handleClickEvent ( const ClickEvent &  evt)
inlinevirtual

Handles the click events when the Snapper is clicked.

It saves its current position as the snap position if the Snapper is pressed. This happens when the drag operation starts.

The snapper will then move to the snap position when the click is released. This happens when the drag operation ends.

Parameters
evtThe click event.
