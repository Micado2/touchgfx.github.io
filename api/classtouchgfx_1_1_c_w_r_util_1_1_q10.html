Detailed Description

Defines a "floating point number" with 10 bits reserved for the fractional part of the decimal number.

Q10 implements some simple arithmetic operations, most yielding a Q10 number and some yielding a Q5 number or a Q15 number as a result.

Q5*Q5=Q10, Q10/Q5=Q5, ...

See also
Q5, Q15
