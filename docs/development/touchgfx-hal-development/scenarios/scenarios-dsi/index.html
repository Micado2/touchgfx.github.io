LTDC+DSI

For MCUs that also have a DSI Host Controller (e.g. STM32F469, STM32F769) then the Tearing Effect and End Of Refresh interrupts should instead be used to transfer parts of- or the the entire framebuffer.

For some application templates we use the End Of Refresh interrupt to set up a new region of the screen to be transferred.
